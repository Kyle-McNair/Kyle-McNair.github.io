Redesigning the Tambopata Website

In the spring and summer of 2020, I was given the opportunity to update and redesign UW-Madison's Geography 339 Assignment website about Tambopata. The goal of this project was to make the website mobile friendly, and to have the proposal maps be condensed into one interactive map. Throughout the project, I was the team's lead developer and made sure to follow best practices in UI/UX design for both desktop and mobile view. Since the release of the site, several professors and organizations worldwide are using this site.

To view the website, click here.

Bootstrap, jQuery, and Leaflet were used to update the website.
